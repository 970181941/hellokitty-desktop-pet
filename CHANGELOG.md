# Changelog

## [1.0.6] - 2026-06-01

### 修复
- 修复 macOS 代码签名验证导致更新安装失败
- ad-hoc 签名无法通过 Squirrel.Mac 验证时，自动回退到浏览器手动下载

## [1.0.5] - 2026-06-01

### 修复
- 修复自动更新下载失败：macOS 构建新增 zip 格式（electron-updater 自动更新必需）
- DMG 保留用于手动安装，zip 用于应用内自动更新

## [1.0.4] - 2026-06-01

### 布局重构
- 侧边面板从右侧移至左侧，始终可见
- 侧边面板默认展示 Hello Kitty 状态信息
- 所有快捷功能（玩耍/日程/待办/设置）统一通过侧边面板展示
- 底部工具栏整合为侧边面板导航（5个视图切换按钮）
- 聊天区域简化为纯聊天界面（header + 状态栏 + 消息 + 输入）
- 窗口固定 730px 宽度（280 面板 + 450 聊天），不再动态缩放

### 技术优化
- 移除窗口 resize IPC 通信（resizeChatWindow）
- 面板切换系统重构为视图切换系统（switchView）
- 清理旧面板互斥逻辑和开关动画代码

## [1.0.3] - 2026-06-01

### 新增
- 状态栏点击展开右侧详情面板（窗口扩展，不遮挡聊天）
- 侧边面板显示主人信息、心情、亲密度进度条、陪伴统计、成就徽章
- 工具栏"状态"按钮统一打开侧边面板

### 动画优化
- 状态栏脉冲提示动画
- 侧边面板卡片交错滑入动画
- 亲密度进度条微光流动效果
- 心情emoji弹跳进入动画
- 工具栏按钮弹性hover效果
- 发送按钮悬浮上提效果
